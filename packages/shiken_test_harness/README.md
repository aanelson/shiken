Helps setup a test harness

## Installation 💻

**❗ In order to start using Test Harness you must have the [Flutter SDK][flutter_install_link] installed on your machine.**

Add `test_harness` to your `pubspec.yaml`:

```yaml
dev_dependencies:
  test_harness:
```

Install it:

```sh
flutter packages get
```
